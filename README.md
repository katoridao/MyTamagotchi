# MyTamagotchi

MyTamagotchi là một trò chơi mô phỏng thú cưng ảo, lấy cảm hứng từ hiện tượng Tamagotchi từng rất phổ biến trên thị trường.

Project được xây dựng với mục đích luyện tập và cải thiện kỹ năng lập trình thông qua việc phát triển một game đơn giản.

---

## Gameplay

Người chơi sẽ chăm sóc một thú cưng ảo bằng cách:

* Cho ăn
* Chơi cùng
* Theo dõi trạng thái (đói, vui, sức khỏe,...)
* Giữ cho thú cưng luôn trong trạng thái tốt

Nếu không chăm sóc đúng cách, thú cưng sẽ dần yếu đi và có thể “game over”.

---

## Features

* Đăng ký, đăng nhập lưu tiến trình chơi thời gian thực
* Hệ thống trạng thái thú cưng (hunger, happiness, health,...)
* Tương tác cơ bản (feed, play, rest,...)
* Trạng thái thay đổi theo thời gian
* Logic game đơn giản, dễ mở rộng
